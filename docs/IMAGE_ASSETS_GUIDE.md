## üìù Tips Generate

2. **Consistency** ‚Äî Pakai kata kunci "Majapahit, Javanese, golden, museum quality" di semua prompt agar style konsisten
3. **Resolution** ‚Äî Generate di resolusi tinggi lalu resize. Lebih baik downscale daripada upscale
4. **WebP conversion** ‚Äî File JPG bisa dikonversi ke WebP nanti untuk optimasi (saya akan bantu)
5. **Naming** ‚Äî Simpan sesuai filename yang tertulis di atas agar langsung cocok dengan kode
